物理化学

本笔记内容主要基于彭笑刚《物理化学讲义》.

第 1 章 物质的能量与经典统计热力学

1.1 单个分子拥有的能量

1.1.2 分子的转动与振动

  • 应用势阱箱模型时隐含了对于分子能够被当作质点处理的假设，实际上随着箱子体积的减小、分子构型的更加复杂，这种近似带来的误差会越来越大，因而需要考虑分子内部结构.
  • 在不破坏化学键的条件下，除单原子分子外一个分子还有转动（rotation）和振动（vibration）两种机械运动模式. 分子的转动不包含有机化学中讨论的分子内转动，后者一般被归类于振动，并且相比正常的振动所需能量较低.
  • 若一个分子有 $N$ 个核，则每个核的运动需要用三个空间坐标描述，即其总的核运动自由度为 $3N$，其中有 $3$ 个自由度用于描述分子的平动（质心的运动）.
    • 对于单原子分子，其总自由度等于 $3$，全部为平动自由度.
    • 对于双原子与线性多原子分子，其有 $2$ 个转动自由度，这是因为绕穿过所有原子核的线性轴的转动不改变核的空间取向. 则其振动自由度的数目为 $(3N-5)$.
    • 对于非线性多原子分子，其有 $3$ 个转动自由度和 $(3N-6)$ 个振动自由度.
  • 需要注意，上述讨论只适用于不考虑分子相互作用的单分子情况，例如理想气体分子. 对于液体和固体中的分子则不可以使用这样的算法.

    • 振动能总是大于 $0$，即分子每时每刻都在振动.
  • 与平动和转动不同，振动既有动能也有势能，一个给定的振动能级中发生着势能和动能的不断转化.
    • 若参与振动的原子核相同，核间化学键越强，化学键提供的回复力就越大、振动频率就越高.
    • 若两种不同分子的化学键键强接近，参与振动的原子越重，振动将进行得越慢、振动频率就越低.
    • 对于同一组原子核，发生弯曲振动比伸缩振动所需的能量更低.
    • 相比平动和转动能级，振动能级的能隙更大，即分子的振动更加难以激发.

1.1.5 分子的电子能级

  • 对电子能级的研究需要求解薛定谔方程，现有数学理论无法对含三个或更多粒子的薛定谔方程精确求解，因此需要给出一些近似条件：
    • 电子运动速度极快，因此考虑电子运动时可以假定原子核静止不动，即定核近似.
    • 在一个分子内，各原子的电子轨道（即波函数）得到不同程度的保持. 对于内层电子尤其如此，它们可以近似为一类“非键轨道”. 原子的外层电子（即价电子）轨道若对称性不匹配，则原子价电子轨道不形成有效的相互作用，形成“非键轨道”；若对称性匹配且相位相同，则形成“成键轨道”；若对称性匹配且相位相反，则形成“反键轨道”.
    • 通过能量最低原理求得各个对称性匹配的原子价轨道线性组合所得的分子轨道，这样求得的分子轨道数目等于原来的原子价轨道数目.
  • 一个电子能级往往对应于多个电子轨道，称为简并电子能级，等能量电子轨道的数目称为该电子能级的简并度.
  • 由定核近似，量子力学对电子能级的解只是表明了分子在平衡核间距上的情形，是一个“静态解”. 分子势能曲线上的每一点都可能对应着一组不同的分组轨道，它们的波函数和本征能级会不同于平衡核间距上对应的分子轨道.
  • 电子-核相互作用对分子能量的贡献，一般要远远大于平动、转动、振动对分子能量的贡献.

1.1.7 分子运动的耦合性

  • 在特定瞬间的状态，由于各种运动的特征时间不同，考虑分子运动的独立性是合理的. 但当考察的时间相比于所涉及的运动形式的特征时间足够长，各种运动之间的耦合就必须考虑.

  • 考虑一个简单的光致电子激发过程和随后的能量转换、能量在各种运动形式上的分配：

    图中显示出一个分子的两个电子能级——基态和一个激发态，以及各自的几个振动能级. 一方面，由于电子能量是分子能量的主要部分，处于电子能激发态的电子能量曲线需要整体向上移动；另一方面，当分子处于电子激发态时，其化学键合强度一般有所降低，因而键长有所增加，电子能量曲线需要整体向右（长距离方向）移动.

    1. 第一步为光吸收和电子激发. 当一个光子的能量和分子的电子能级差相匹配且满足光谱选律时，分子可吸收这个光子而使得发生电子激发，其过程非常迅速以至于分子构型在这一瞬间尚没有变化，这在图中体现为电子的垂直跃迁. 实验表明分子进入电子激发态的第三振动激发态的概率最高.

    2. 第二步为振动去激发. 第一步发生后，分子处于振动激发态，实验表明在同一电子能级内部很容易发生振动去激发，分子将通过外部去激发（红外辐射）或内部去激发（即将能量转化为更容易激发的转动和平动能量）的途径达到电子激发态的振动基态.

    3. 第三步为电子去激发. 分子到达电子激发态的振动基态后还可以进一步耗散能量回到电子激发前的状态，这可以通过发射光子或将能量转化为振动、转动、平动能量而消耗掉的途径实现，同样地这是一个迅速的过程，因而体现为垂直跃迁.

    4. 第四步为振动去激发. 这与第二步是类似的.

  • 实验表明，一旦发生电子激发，整个过程将自发完成；反过来，由平动、转动、振动激发导致的电子激发是很难发生的. 这说明如果某种能量类型具有高的相邻能级差，就能自发地转化为具有低的相邻能级差的能量类型.

    • 基于此，给出一种分子能量类型的基本能隙定义为：某种给定运动形式的量子基态与第一激发态之间的能量差称为该运动形式的基本能隙.

  • 从低基本能隙运动引起高基本能隙运动激发的过程并非不可发生，实际上常温常压下的许多化学反应就是需要通过能量的“累积“而实现这一过程，这也是化学动力学中”活化能“概念的分子基础. 低基本能隙的能量形式必须有固定的方向、有一定的组织，才能使这样的转换易于发生.

1.2 多分子系统

1.2.1 统计热力学的基本思想

  • 统计热力学是完全基于原子、分子的定量研究的学科，它充当着连接单分子世界（量子力学）和宏观世界（经典热力学）的桥梁.
  • Boltzmann 所开创的经典统计热力学在融入了量子力学理论之后发展成为量子统计热力学，许多现象得到了完美的解释.
  • 在给定的外界条件下，一个化学系统总是可以到达一个稳定状态，称之为宏观平衡态，且一个多分子系统一本只有一种固定的全域平衡态，即宏观平衡态具有唯一性.
    • 后续还会涉及到局域平衡态的讨论.
    • 平衡态的意义在于：
      • 和外部世界达到了平衡的系统状态
      • 系统内可观测的物理量、化学量在系统内部各处都不随时间变化的状态.（这里假设了系统足够大从而可以暂时不考虑系统和环境、系统内部的界面部分的不均匀性，而在界面效应称为重要因素的时候则要加以特殊处理）
    • 需要注意，对于一个给定的宏观状态，一个分子可能经历着多重分子量子态，因为分子的各种运动都在持续进行.
  • 在一个给定瞬间，每个分子可以用一组量子数来描述其状态. 一个多分子系统对应的一套多重量子数，称为该系统的微观结构，它一般远多于系统的分子数. 对于某个特定的宏观状态，对应的微观结构数量称为该宏观状态的权重（weight），记为 $W$.
  • 一个系统可以有许多宏观状态，而系统的一个宏观状态对应于多重微观结构，微观结构之间可以通过分子-分子、分子-外部边界的相互作用进行转化，宏观状态的转变同样也是由微观结构的转化实现.
    • 对于具体问题，可以选择合适的可观测量来描述系统的一个特定宏观状态.
  • 宏观平衡态的条件可以适当放松，只要求宏观可观测量的内在稳定性或进一步简化为各处均匀性.
  • 这一思想代表性的实验验证为对于 Brown 运动的研究，Einstein 和 Smoluchowski（斯陌路切斯基）提出的使用统计热力学方法处理的解释得到了 Perrin（佩云）的实验证实，并测得了高精确度的 Avogadro 常数，科学界开始普遍接受原子论和统计热力学.
  • 统计热力学基本假定：对于一个给定的系统，满足边界条件的所有可能宏观状态的所有微观结构出现概率相等.
    • 可以理解为这个基本假定的核心是“等可能性”.
  • 其他的普适性假定：
    • 假定一：在任何宏观可分辨的时间范围内，一个系统可以多次得到所有不同的可能微观结构，这是因为相比于宏观可测时间，微观结构之间的变化非常快. 这个假定的意义在于某些宏观可观测量的实验值可以通过其所有可能微观结构中的相应量子参数的统计平均值求得.
    • 假定二：对于单分子或者少数几个分子的系统，可以用该系统的时间平均值代表多个宏观上等价单分子系统的统计平均值. 这个假定的意义在于可以将统计热力学的方法用于单分子系统，另外这也是 Gibbs 统计理论的基础.

1.2.3 最可几宏观状态

  • 具有最大微观结构数，即最大权重的宏观状态称为最可几宏观状态（最可几分布）.
  • 对于一个系统，不同宏观状态之间的权重相差很大，在足够的时间后系统最终都会在最可几宏观状态附近涨落，对于实际系统这样的涨落不具有宏观上的可观测性.
  • 若系统受到扰动离开最可几宏观状态，也会在宏观可观测时间内回到最可几宏观状态.
  • 对于孤立系统，系统与环境不发生任何相互作用，则最可几宏观状态是唯一的，即宏观语言描述的平衡态.
  • 通过计算可以知道即使系统的粒子数不太大，权重分布也可以相当好地近似为单一分布，一旦稍微偏离，相应宏观状态的权重就可以忽略不计，即处于稳态的典型宏观状态，可以很好地只用它的最可几宏观状态来描述.

1.2.4 熵

  • 总结以上讨论，当我们在提及“稳定”这一概念时，实际上是在谈论一个宏观状态的微观结构数的多寡，如果一个宏观状态的微观结构数远远大于另一个，则从统计的观点看，由第一个状态到第二个状态地变化是不容易发生的，即第一个状态比第二个状态稳定。宏观系统中发生的过程具有一个确定的方向，因此可以明确地比较宏观状态之间的稳定性差别.

  • 对于宏观系统中发生的过程的研究，热力学给出了一个宏观的概念——熵，它是一个广度性质，系统的熵值正比于系统的大小，即熵具有加和性.

1.3.8 能量最低原理的适用性

  • 由 1.3.7 的讨论可知，能量最低原理的近似对于电子能级、高频振动能级有较好的适用性，而对于转动与平动则不然.
  • 到目前为止对统计热力学的讨论显示出 $kT$ 和 $RT$ 是具有特殊物理化学意义的量，它们是分子能级的标度，其量纲与能量一致.
  • 以相关的量子基态作为零点，当一个激发态的能量远小于 $kT$ 时，分子在此能级上的分布与基态相似，此时激发态将对热能有明显的贡献.
  • 而相对于相关量子基态，若分子某能量类型的基本能隙远大于 $kT$ 时，其激发态将基本不会被分子所占据，相应地它对热能的贡献可以忽略不计，此时能量最低原理是适用的.

1.3.9 从配分函数到熵

  • 不同于求解系统宏观能量的过程，一个具体微观量子态的熵是没有意义的，因此对此求统计平均值是无效的. 合理的方法是利用 Boltzmann 公式将熵与宏观状态权重联系起来，利用 Boltzmann 分布及其对应配分函数来反映系统平衡态（最可几分布状态）下熵的情况.

1.4 本章总结

  以上的“第一章”内容在我所参考的《物理化学讲义》中是三章的内容，但由于它们的主题具有非常好的连贯性，我将它们合成为了一章，到此可算是补上了一个初学物理化学时的遗憾——未能建立起对统计热力学的一个完整认知. 趁热打铁总是好的，故而写一写这五天（10月4日-10月8日）的学习心得.

  在国内传统意义上的物理化学课程中，量子与结构化学的内容通常不在讲授范围内，统计热力学的内容也多为选学，或也干脆不纳入到物理化学的教学体系之中. 这就导致我们的物理化学必然要从宏观热力学讲起，一方面忽略了微观结构的基础，另一方面对于宏观热力学的处理又显得“物理味”太浓，使学生难以将其用在化学中施展拳脚. 私以为彭的处理——将量子与结构化学、统计热力学中一些基本的概念作简要介绍，既不使物理化学陷于完全抽象的量子力学讨论中，又为下一步在分子层面（化学研究的最基本对象）上理解热力学打下了基础，至少在学习体验上，这是一次非常优秀的尝试.

  本章的逻辑主线是十分清晰的：从我们最熟悉的“能量”讲起，同时用抽象的数学语言和具象的图像语言说明了在微观世界中能量的量子化，给出了单个分子几种运动形式以及电子、核所具有的能量的描述；从单分子到多分子，我们需要引入“统计平均”的方法，正是统计热力学将量子力学和宏观热力学得以连接起来，它的核心是 Boltzmann 方程；在对宏观状态权重的讨论中，导出了一个非常有用的工具——配分函数，它使得我们可以利用分子的微观特征来分析系统的宏观可观测量，于是我们最终得以将前面所提到的“能量”和“熵”在分子层次上定量化.

  回顾在本章的讨论中所用到的一些核心概念：

  • 分子运动的独立性、分子能量的加和性、分子配分函数的乘积性：除此之外还包括微观状态数对于不同能级的乘积性等等，它们是一脉相承的. 它们首先给了我们一个能够将分子系统的各类型能量、熵进行逐一分析的理论基础，也为我们一路上的数学推导提供了最基本的合理假定：哪些量能够求和、哪些量能够求积.
  • 微观结构、宏观状态、最可几分布：统计热力学的核心就是从微观世界过渡到宏观世界，我们最需要回答的一个问题是：怎样确保我们讨论的宏观性质正好是某一种状态下的微观结构所确定的呢？这要靠实验测定来证明，其结果是，对于超大量分子数的典型宏观系统，存在一个决定性的“最可几分布”，处于最可几宏观状态的系统所表现的性质可以代表系统的宏观性质. 追溯到微观结构上，就用到了 Boltzmann 分布.
  • 能量、熵、温度：这三个在热力学中占据了核心地位的概念在本章中的引入过程是十分精彩的. 我们将关注的重点放在了其中真正客观存在的两个事物之上：能量与熵，温度则作为二者之间的数学导出关系来定义，从而构建出一个类似运动学中“位移-时间-速度”的概念体系. 这是因为温度是一个控制性的条件，而不像能量与熵一样，更多地与物质本身的结构和性质相关联. 另外，统计热力学对于这三个概念的处理还不够完整，需要在热力学中进一步探究.

  接下来，谈谈本章内容的局限性：

  • 如作者所说，统计热力学在化学上的应用还不够深入，或者说，仍然存在一些理论上处理的困难. 例如多数结论仍然要求“理想气体分子”或“室温”等条件，这可以说是为了推导出有实用意义的结论所作的妥协，这些困难主要来源于系统中巨大的分子数量以及量子力学中能级分立所带来的微积分处理的误差.
  • 许多理论仍然是建立在对单一组分系统之上的，虽然考虑到了“浓度”这一概念的定义及其合理性，但真正将统计热力学用在处理溶液体系上还有很多问题需要解决.
  • 本章内容并不是一个完全基于量子力学的讨论，我们所用的方法基本上是属于经典统计热力学的，即 Boltzmann-Maxwell 统计，而真正的量子统计—— Bose-Einstein 统计和 Fermi-Dirac 统计则没有提及，这方面的内容可以参考傅献彩版《物理化学（第5版）》，其中有相关的介绍.
  • 本章的讨论中用到的最基础的假设便是分子运动可以相互独立、分子间的相互作用一定程度下可以忽略，而实际系统中未必如此，尤其是后者. 为了解决这样的问题，还需要使用系综统计的方法.

  最后，笔者还希望在未来能够看到统计热力学和宏观热力学的方法在环境科学中得到广泛应用，并在环境科学基本理论体系的建构上发挥关键性的作用. 而在此之前，首先要着眼面对的就是统计热力学与宏观热力学理论在多组分的复杂系统中的应用，以及此过程中所遇到的数学理论、计算方法上的困难.
